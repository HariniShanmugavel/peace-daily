PeaceDaily is a mobile application built with React Native (frontend) and FastAPI (backend) to help users track and improve their mental health through daily check-ins. Users can log stress, energy, anxiety, and focus levels, view trends, and access helpful resources.

Features

Daily mental health check-in (stress, energy, anxiety, focus)

Visual progress tracking

Quick access to mental health tips and resources

Dark mode toggle

Interactive UI for easy navigation

Backend powered by FastAPI with database support

Tech Stack

Frontend: React Native, Expo, TypeScript

Backend: FastAPI, Python

Database: SQLite (or PostgreSQL/MySQL if upgraded)

State Management: React hooks

Version Control: Git/GitHub
